Welcome

to the e-zine for experimenters in Processing and Arduino

Processing is an open-source programming language and IDE designed for electronic arts and visual design communities.

Arduino is an open-source electronics prototyping platform designed for artists, designers, hobbyists, and anyone interested in making interactive objects or environments.

Being able to communicate from Processing to an Arduino is very useful. For instance, Processing allows for more input devices, such as a keyboard or microphone, to communicate to the Arduino by using serial connection.

Processing and Arduino were meant to be. However, it's important that you learn the basics for each platform before you use them in conjunction. This website is an aggregate resource for Processing and Arduino projects, designed to be an e-zine style reference guide. It features handpicked tutorials, an exhibition of inspirational projects, important hardware components, specialized stores, and a list of communities to join.
